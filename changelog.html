Version 1.5.0

Released on May 30, 2025

Latest Release

New Features

  • Enhanced AI Content Assistant

    Added support for context-aware content suggestions and improved translation capabilities for 5 additional languages.

  • Image Optimization Pipeline

    Automatically optimize and compress images before committing to your repository. Supports JPEG, PNG, and WebP formats.

  • Advanced Merge Conflict Resolution

    New visual interface for resolving Git merge conflicts directly from your mobile device.

Improvements

  • Faster Markdown Rendering

    Optimized preview engine now renders Markdown 60% faster than previous versions.

  • Enhanced Dark Mode

    Improved contrast and readability in dark mode across all screens.

  • Battery Usage Optimization

    Reduced background processing to improve battery life by up to 25%.

Bug Fixes

  • Fixed OAuth token refresh issue with GitHub

    Resolved authentication errors when tokens expired during active sessions.

  • Corrected image upload path handling

    Fixed an issue where image paths were incorrectly formatted when using certain folder structures.
